# Rufumleitung einrichten

So leiten Sie Ihre Geschäftsnummer an den BureauFlow KI-Assistenten weiter.

## Empfohlene Einstellung

Wir empfehlen die **bedingte Rufumleitung bei Nicht-Erreichbarkeit**. So können Sie weiterhin selbst Anrufe annehmen — der Assistent springt nur ein, wenn Sie nicht rangehen.

## Anleitung nach Anbieter

### Deutsche Telekom
```
Aktivieren:   **61*IHRE_BUREAUFLOW_NUMMER**
Deaktivieren: ##61#
```

### Vodafone
```
Aktivieren:   **61*IHRE_BUREAUFLOW_NUMMER#
Deaktivieren: ##61#
```

### O2 / Telefonica
```
Aktivieren:   **61*IHRE_BUREAUFLOW_NUMMER#
Deaktivieren: ##61#
```

### Festnetz (Fritzbox)
1. Öffnen Sie die Fritzbox-Oberfläche (fritz.box)
2. Gehen Sie zu **Telefonie → Rufumleitung**
3. Klicken Sie auf **Neue Rufumleitung**
4. Wählen Sie "bei Besetzt/nicht erreichbar"
5. Geben Sie die BureauFlow-Nummer ein

## Testen

Nach der Einrichtung: Rufen Sie Ihre eigene Nummer an und legen Sie nicht auf — nach ca. 20 Sekunden sollte der BureauFlow-Assistent antworten.
